# TekxLobster

A 2D arcade fighting game built with HTML5 Canvas and JavaScript.

**[Play it here](https://123mikeyd.github.io/TekxLobster/)**

## Characters

- **TEKNIUM** -- The AI Sage. Energy blast special attack.
- **LOBSTER** -- Claw Fighter. Beam cannon special attack.

## How to Play

Open `index.html` in any browser. No install, no dependencies.

### Controls
| Action | Key |
|--------|-----|
| Move | A / D |
| Jump | W |
| Crouch | S |
| Block | Q |
| Light Attack | U |
| Medium Attack | I |
| Heavy Attack | O |
| Special (costs 50 energy) | P |

## Features
- Character select screen
- Single player vs CPU AI
- Best of 3 rounds with round timer
- Health bars + energy meters
- Pixel art sprite sheets
- Web Audio sound effects (no files needed)
- City night stage with animated crowd

## Built With
- Pure HTML5 + Canvas + JavaScript (single file)
- No frameworks, no build tools
- Sprite sheets generated with AI, chroma-keyed in-browser
